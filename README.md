### 一、相关技术
- 后端：`Java`、`JavaWeb` / `Springboot`。

- 数据库：`MySQL`

### 三、功能描述
系统分为（身份）：学生、企业、管理员。

**学生用户功能：**
1. 登录
2. 注册
3. 用户管理
4. 三方协议书
5. 就业政策
6. 就业服务管理

**企业用户功能：**
1. 登录
2. 用户信息
3. 就业政策
4. 教师模块功能分为三方协议书、统计信息、用户管理

**管理员功能：**
1. 登录
2. 用户管理
3. 学院信息管理
4. 就业模块功能分为就业管理、就业服务管理、统计信息、用户信息
